Conceptos básicos

Template

HTML con anotaciones especiales, también conocido como partial

Directive

Extiende el HTML con atributos y elementos personalizados

Model

Los datos mostrados y con los que el usuario tiene interacción

Scope

Contexto donde se almacena el modelo para ser accesado por controladores, directivas y expresiones

Compiler

Analiza la plantilla e instancia directivas y expresiones

Filter

Da formato al valor de una expresión para mostrarla al usuario

View

Lo que el usuario ve, puede ser la plantilla

Data binding

Sincronización automática entre el modelo y la vista - two-way data binding-

Cualquier cambio en la vista es reflejado de inmediato en el modelo y viceversa.

El modelo es la única fuente confiable del estado de la aplicación. Ya que la vista es la proyección del modelo, el controlador está aislado de la vista.

Controller

Lógica de negocio detrás de la vista

Dependency injection

Crea y liga objetos y funciones

Injector

Contenedor de injección de dependencias

Module

Contenedor para distintas partes de una app, útil para la agrupación de elementos de la misma índole o que están muy relacionados entre sí

Service

Lógica de negocio reusable, independiente de la vista
